# 📊 Machine Learning Regression Implementations

This repository contains hands-on implementations of Simple Linear Regression and Multiple Linear Regression using Python and Jupyter Notebooks.
It is designed to help beginners build a strong foundation in Machine Learning by understanding concepts step-by-step with practical examples.

# 📌 Project Overview

This project focuses on understanding how regression models work, starting from basic concepts to slightly advanced implementations.

🔹 Simple Linear Regression
Works with one independent variable

Learns relationship using a straight line:

y = mx + c

🔹 Multiple Linear Regression
Works with multiple input features

General form:

y = b0 + b1x1 + b2x2 + ... + bn*xn
